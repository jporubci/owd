On-Web Diagnostics Tool

Login | Register

Description

We wish to measure car statistics such as engine load, coolant temperature, speed, and RPM and store them in an online database.

Usefulness

Monitor car health both with real-time updates and post-data analysis (i.e., read codes and produce the most likely cause, tracking trends over time, per user profile).

Realness

The data would be real-time car information obtained from the OBD-II tool.

Functionality

Each user would have their own username and password, and a user_id will be assigned as the primary key.

Basic Functions

  • A way for Users to login to their generated profile or create a new profile.
  • Users have the following privileges:
    • Upload own data
    • View own data
    • Delete own data
  • Admins have the following privileges:
    • Delete users

Advanced Functions

  • Real-time uploading on estimated MPG: engine load, coolant temperature, speed, RPM.
  • Comprehensive Vehicle Diagnostic Analysis
